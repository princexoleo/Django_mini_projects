# Django Authentication 

# Features
* Custom user authentication
* Social Authentication 
  * Github Authentication
  * Google Authentication
* User profile

# Run
Before setup this project,  create a virtualenv and install the required libriaries by `pip install -r requirements.py`. 
please remove the `_copy` from `dj_auth/settigs_copy.py` file into `settings.py` and set the django key in the 'settings.py`.
